Why Use an API?

APIs are useful because they allow a website to interact with a database of information so a user can get up to date data without reloading a webpage

There are 4 main actions APIs can take and these are:

GET requests which are requests made to retrieve information from the server

POST which sends changes from client to server, in other words creating a new entry.

PUT which revises or adds to existing information rather than creating new information.

DELETE which deletes existing information from a database. Take a look at the examples to show GET, POST, PUT and DELETE in action
